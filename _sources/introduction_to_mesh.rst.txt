An introduction to Mesh
---------------------------------------

Mesh is a data management system based on object modelling. Mesh is used for efficiently managing time series data and related information used in different work processes.

To be able to work efficiently with Mesh it is important to have an understanding of the basic :doc:`concepts of Mesh <mesh_concepts>` and the software architecture consisting of the :doc:`Mesh server <mesh_server>` and a :doc:`Mesh client <mesh_client>`.
