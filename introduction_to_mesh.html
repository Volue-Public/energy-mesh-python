An introduction to MeshÂ¶

Mesh is a data management system based on object modelling. Mesh is used for efficiently managing time series data and related information used in different work processes.

To be able to work efficiently with Mesh it is important to have an understanding of the basic concepts of Mesh and the software architecture consisting of the Mesh server and a Mesh client.
