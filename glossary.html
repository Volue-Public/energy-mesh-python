Glossary¶

API¶

Application Programming Interface.

db¶

Database.

IDE¶

Integrated Development Environment. Software that consolidates the basic tools needed for software testing and writing.

Mesh functions¶

Used to perform tasks and calculations on data in the Mesh model.

Mesh model¶

Represents the physical infrastructure of a customers area and assets.

Mesh search language¶

A language with a custom syntax designed to traverse the Mesh model.

SDK¶

Software Development Kit. A collection of software development tools in one installable package.

Time series¶

A sequence of data points that occur in successive order over some period of time.

venv¶

Virtual environment. Concept for working with separate versions of Python and/or Python packages.
