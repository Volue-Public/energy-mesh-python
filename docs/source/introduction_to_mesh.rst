An introduction to Mesh
---------------------------------------

Mesh is a data management system based on object modelling. Mesh is used for efficiently managing time series data and related information used in different work processes.

To be able to work efficiently with Mesh it is important to have an understanding of the basic concepts of Mesh. More specifically the :doc:`Mesh model <mesh_model>` and the software architecture consiting of the :doc:`Mesh server <mesh_server>` and a :doc:`Mesh client <mesh_client>`.
